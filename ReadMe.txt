I have use pandas library which is a powerful tool for data analysis and manipulation.
Here I had generate a final payment sheet from the given data sheets of different orders.

Run:

In order to run the code we must first create a python virtual environment using command:
"python3 -m venv virtual_env_name".

Then install all the files from requirements.txt using command:
"python3 -m pip3 install -r requirements.txt".

Then run the main file using command:
"python3 payment.py".
